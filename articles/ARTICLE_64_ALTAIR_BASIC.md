# Altairの実機を一度も見ずに完成した——8週間と4KBが産んだソフトウェア産業の産声

## はじめに

GitHubの片隅に「産声」のコードを見つけた。

**Altair BASIC**。1975年、Bill Gates と Paul Allen が8週間で書いた4KBのBASICインタプリタ。

舞台はハーバード大学 Aiken Computation Lab の PDP-10。Altair 8800の実機は一度も手元になかった。彼らは Allen が書いた8080エミュレーターをPDP-10上で動かし、仮想の Altair の上でコードを書き続けた。

8週間後、Paul Allen は完成したテープを持ってアルバカーキに飛んだ。最終着陸態勢に入ったとき、「**ブートローダーを書き忘れた**」と気づいた。テープをAltairのメモリに読み込むプログラムがなければ、デモは始まらない。Allenは8080の機械語コードを頭の中で組み立てながら、着陸前に完成させた。

MITS創業者 Ed Roberts の前でデモが始まり、Altairが「**MEMORY SIZE?**」と表示した瞬間——それが、Gates と Allen が初めて自分たちのコードが実機で動いているのを見た瞬間だった。

1年後、Bill Gates は「ホビイストへの公開書簡」を書いた。**「あなたたちの大半はソフトウェアを盗んでいる」。** それが商業ソフトウェア産業の産声だった。

## 発掘された痕跡

### 痕跡1：バイト節約ジャンプ——1バイトのために全てを犠牲にする

```asm
SNERR:  MVI E,ERRSN
        DB 01H          ; LXI B命令として次の2バイトを「飲み込む」
DV0ERR: MVI E,ERRDV0
        DB 01H          ; 同じトリック、別エントリーポイント
NFERR:  MVI E,ERRNF
ERROR:  CALL STKINI     ; 3つのエラーパスが全部ここに収束
```

3つのエラーハンドラー（構文エラー・ゼロ除算・数値フォーマットエラー）が、明示的な `JMP` 命令を使わずに1つの出口に収束する。

`DB 01H` は `LXI B` 命令の1バイト目。この1バイトは「次の2バイトをオペランドとして無視する」という効果を持つ。つまり直後の `MVI E,xxx` の2バイトは **命令ではなくデータとして読み飛ばされる**。

`JMP` 命令3バイト×3本分 = **9バイトを節約。** 4KBのうちの9バイト。

**1バイトのために思考の限りを尽くす。それが1975年のプログラミングだった。**

### 痕跡2：RSTベクタの横領——割り込みをホットルーティンに転用

```asm
; 8080の割り込みベクタを通常用途に再利用
RST 1   →  SYNCHK   ; 構文チェック（最頻出ルーティン）
RST 2   →  CHRGET   ; 文字フェッチ＋フラグ自動操作
RST 4   →  COMPAR   ; 16ビット比較
RST 5   →  FSIGN    ; 浮動小数点符号テスト
```

8080 CPUは8本のハードウェア割り込みベクタスロットを持つ（各8バイト）。Altair BASIC はハードウェア割り込みを全て無効化し、**64バイトのベクタ領域を「頻繁に呼ばれる内部ルーティンの置き場」として横領した。**

通常の `CALL` 命令は3バイト。`RST n` 命令は1バイト。**2バイトの節約が積み重なる。**

割り込みを捨てることで性能を買う——これが「4KBという牢獄の中での生存術」だった。

### 痕跡3：Monte Davidoff の浮動小数点——標準のない時代の自力設計

```
32ビット浮動小数点フォーマット（Microsoft Binary Format）:

ビット 31-24: 8ビット指数（バイアス128）
ビット 23:    1ビット符号（0=正, 1=負）
ビット 22-0:  23ビット仮数部

数値例:
  10  → 84h, 20h, 00h, 00h
   1  → 81h, 00h, 00h, 00h
 0.5  → 80h, 00h, 00h, 00h
```

1975年、IEEE 754は存在しなかった（制定は1985年）。浮動小数点の業界標準は何もなかった。

Gates と Allen が「浮動小数点ルーティンが書けない」と悩んでいるのを夕食の席で聞きつけたのが、ハーバードの学部生 **Monte Davidoff**（当時19歳）。「以前書いたことがある」と申し出て、**400ドルで受注した。**

Davidoff が設計したこのフォーマットは **Microsoft Binary Format（MBF）** として全ての8ビット Microsoft BASIC に継承された。GW-BASIC、QuickBASIC 4.0以前まで——**20年近く生き続けた。**

### 痕跡4：CRUNCH——キーワードを1バイトトークンに圧縮する瞬間

```
ユーザー入力: FOR I=1 TO 10   (13バイト)
         ↓ CRUNCH処理
トークン化:  [82h][I][=][1][94h][10]  (7バイト)
              ↑FOR                ↑TO

キーワードテーブル（一部）:
FOR=82h, NEXT=83h, DATA=84h, INPUT=85h,
GOTO=89h, RUN=8Ah, IF=8Bh, PRINT=91h,
TO=94h, THEN=96h, NOT=98h, AND=99h
```

ユーザーがBASICコードを入力した瞬間、CRUNCH ルーティンが走る。キーワード文字列を **7ビット目が立ったシングルバイトトークン** に変換。

2つの問題を同時に解決：
1. **メモリ上のプログラムサイズ削減**（文字列→1バイト）
2. **実行時の解析高速化**（文字列比較不要、テーブルオフセット直読み）

ただし `"` の中と `DATA` 文の中ではトークン化を無効化する処理も含む——約100行のルーティンが、入力の瞬間に二役をこなす。

### 痕跡5：着陸直前に書いたブートローダー

Paul Allen がアルバカーキ行きの飛行機に乗ったのは1975年1月。8週間の開発が終わり、完成したパンチテープを持っていた。

最終着陸態勢に入ったとき、気づいた。

**テープをAltairのメモリに読み込むブートローダーを書いていない。**

Allenは8080の機械語コードを頭の中で組み立てながら、着陸前にブートローダーを完成させた。Ed Roberts の前でデモが始まり、Altairが「MEMORY SIZE?」と表示した。

それが **Gates と Allen が初めて実機で自分たちのコードを見た瞬間** だった。8週間、実機なしで書き続けたエミュレーターが完璧に正確だったことの証明でもあった。

### 痕跡6：「ホビイストへの公開書簡」——商業ソフトウェア誕生の瞬間

```
An Open Letter to Hobbyists
William Henry Gates III, February 3, 1976

"As the majority of hobbyists must be aware, most of you steal
your software."
（大多数のホビイストが認識しているように、あなたたちの大半は
ソフトウェアを盗んでいる。）

"The amount of royalties we have received from sales to hobbyists
makes the time spent on Altair BASIC worth less than $2 an hour."
（ホビイスト向け販売から得たロイヤリティは、Altair BASIC開発に
費やした時間を時給2ドル以下にしてしまっている。）
```

1975年6月、パロアルトのセミナーでテープが流出した。Dan Sokol がホームブリュー・コンピュータ・クラブの集会に50本のコピーを持ち込み、「1本持って行ったら2本返せ」と配布した。

8ヶ月後、Gates は書いた。

「誰がタダでプロの仕事をできるのか？どのホビイストが3人年のプログラミング、バグ発見、ドキュメント作成をして無料で配布できるというのか？」

これを読んだホビイストたちは激怒した。「ソフトウェアは自由に共有するものだ」という文化への真っ向からの挑戦だった。しかし **この書簡が商業ソフトウェア産業の産声だった。**

## 年表

**1974年12月**: Popular Electronics 誌がAltair 8800を表紙に掲載。Gates と Allen が見て「作れる」と確信。

**1975年1月〜3月**: Harvard の PDP-10 で8週間の開発。Allen が8080エミュレーターを書き、Gates が BASIC インタプリタを書く。浮動小数点は Davidoff が担当（400ドル）。

**1975年1月下旬**: Allen がアルバカーキへ。着陸直前にブートローダーを完成させる。デモ成功。MITS との契約締結。

**1975年6月**: MITS セミナーでテープ流出。ホームブリュー・コンピュータ・クラブで複製が配布される。

**1976年2月3日**: Gates、「ホビイストへの公開書簡」を発表。商業ソフトウェアの権利を主張。

**1975年〜1977年**: 4K版、8K版、Extended版をリリース。TRS-80、Apple II、Commodore PET など主要マイコンに採用。

**1980年**: IBM PC への採用に向けて交渉開始。Altair BASIC の成功が IBM との契約への信頼を生む。

**1981年**: IBM PC BASIC リリース。MS-DOS と組み合わせてMicrosoftが世界制覇へ。

## 鑑定人所見

Altair BASIC は「産声」だ。

4KBという牢獄の中で、実機のない8週間が過ぎた。Allen は着陸直前にブートローダーを書き、Ed Roberts の前でAltairが「MEMORY SIZE?」と問いかけた——その瞬間が、商業ソフトウェア産業の産声だった。

最も象徴的なのは **「ホビイストへの公開書簡」** だ。「ソフトウェアは自由に共有するもの」というホームブリューの文化に、20歳の Gates が「違う、これは商品だ」と言った。ホビイストたちは激怒した。しかしその主張が、100兆円市場の扉を開いた。

Davidoff の浮動小数点は400ドルで買われ、20年間 Microsoft BASIC の基盤になった。バイト節約トリックの一つ一つが、極限状態の思考の結晶だ。**4KBという制約が、贅肉を一切許さない美しさを生んだ。**

40年後、Gates はMicrosoftのCEOを引退し、世界最大の財団を運営している。しかし全ての始まりは、Harvard の PDP-10 で、Altairの実機を見ずに書いた4KBだった。

**産声は小さかった。しかし世界を変えた。**
